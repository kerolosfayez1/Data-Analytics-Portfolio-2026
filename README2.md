# 📊 Predictive Modeling for Employee Retention

**Developed by: Kerolos Fayez**

## 📝 Project Overview
This project aims to analyze employee data to predict attrition (why employees leave). Using the IBM HR dataset, I explored various factors like salary, distance from home, and work-life balance to build a predictive model that helps organizations improve their retention strategies.

## 🛠️ Key Skills Applied
* **Data Cleaning:** Handling missing values and encoding categorical data.
* **Exploratory Data Analysis (EDA):** Visualizing correlations between job satisfaction and attrition.
* **Machine Learning:** Building classification models to predict employee turnover.

## 📈 Business Impact
Providing HR departments with data-driven insights to proactively address employee concerns and reduce turnover costs.
