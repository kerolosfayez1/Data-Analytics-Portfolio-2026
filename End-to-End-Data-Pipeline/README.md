# 🖤 End-to-End Data Engineering & Analysis Pipeline

**Developed by: Kerolos Fayez | Intro Code**

## 📝 Project Overview
This project demonstrates the full lifecycle of data analysis. I started with raw sales data, processed it using Python (Pandas) for cleaning, and visualized the insights in a high-impact Power BI dashboard.

## 🛠️ Project Components
* **Python Script:** For data cleaning and handling missing values.
* **Power BI Dashboard:** An interactive "Dark Mode" dashboard for sales insights.
* **Data Source:** Raw sales datasets.
