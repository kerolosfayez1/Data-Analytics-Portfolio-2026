# 📈 Sales Forecasting using Machine Learning

**Developed by: Kerolos Fayez**

## 📝 Project Overview
This project involves building a time-series forecasting model to predict future sales trends. By applying Machine Learning techniques, I analyzed seasonal patterns and historical data to help businesses plan their inventory and marketing strategies more effectively.

## 🛠️ Key Skills Applied
* **Time-Series Analysis:** Working with dates, trends, and seasonality.
* **Feature Engineering:** Creating lag features and rolling window statistics.
* **Regression Modeling:** Using ML algorithms to predict continuous numerical values.

## 📈 Business Impact
Accurate sales forecasting helps in reducing waste, managing workforce distribution, and maximizing revenue during peak seasons.
