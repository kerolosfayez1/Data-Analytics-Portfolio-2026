# 🎯 Customer Behavior Segmentation using Machine Learning

**Developed by: Kerolos Fayez**

## 📝 Project Overview
This project focuses on grouping customers based on their purchasing behavior using clustering algorithms (K-Means & DBSCAN).

## 🛠️ Key Skills Applied
* **Unsupervised Learning:** K-Means & DBSCAN.
* **Data Visualization:** Seaborn & Matplotlib.
