# 🍊 Interactive Sales Insights Dashboard

**Developed by: Kerolos Fayez | Intro Code**

## 📝 Project Overview
This professional dashboard was designed to provide a comprehensive view of sales performance. Using a vibrant orange-themed UI, the dashboard simplifies complex data into actionable insights for stakeholders, focusing on revenue trends and product distribution.

## 🛠️ Key Features
* **Advanced Visualizations:** Utilizing custom charts to track monthly sales growth.
* **Interactive Filters:** Users can drill down by region, product category, and time periods.
* **KPI Metrics:** Focused on Total Sales, Profit Margin, and Top 5 Performing Products.

## 🚀 Tools Used
* **Power BI:** For data modeling and visualization.
* **DAX:** Used for creating complex measures and calculated columns.
* **Power Query:** For data cleaning and transformation.
